<h1 align="center">SwiftClicker</h1>

Finally, a modern, lightweight, and free autoclicker for macOS. Built in Swift with a focus on speed, simplicity, and great design.

## Requirements

- macOS 14 Sonoma or later

## Installation

1) Go to the [latest release](releases/latest)
2) Download the .dmg file
3) Drag ```SwiftClicker.dmg``` into the ```Applications``` folder, as directed
4) Open **SwiftClicker**  

_**Note: Since I don't have a paid Apple Developer account, you will need to head into Settings > Privacy & Security, then scroll down and allow ```SwiftClicker.app``` to open**_

## Granting Accessibility Access

SwiftClicker needs Accessibility Access to autoclick. In order to grant access, simply click **Grant Accessibility Access** in the app, then follow the system prompts. After you've granted access, quit the app and reopen to start using it.
